# Desafio Backend

## Running API

```bash
git clone https://github.com/lucaspaiva1/desafio-backend.git
cd desafio-backend
cp .env.example .env
npm i
npm start
```

#### Seed database

```bash
npm run seed
```

## Dependencies

- MongoDB
- node 14.x

## Online Version

https://desafio-coteminas.herokuapp.com/

Using heroku to host the API and database with MongoDB cloud.
